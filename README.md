# Simple Port Scanner

## 📌 Overview
This project is a basic port scanner written in Python.  
It scans a target IP address to check which common ports are open.

The goal of this project is to understand how network ports work and how devices communicate over a network.

---

## ⚙️ Features
- Scans common ports (21, 22, 23, 25, 53, 80, 443)  
- Detects open ports on a target system  
- Simple and easy-to-use script  

---

## 🛠️ Technologies Used
- Python  
- Socket Programming
