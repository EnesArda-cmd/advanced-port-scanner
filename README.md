# Mini Port Scanner Project

This is a simple port scanner built with Python to demonstrate basic network security skills.

## Features
- Scans common ports on a target IP
- Identifies which ports are open
- Fast and lightweight script
- Works on Windows (Python required)

## How to Use
1. Run `port_scanner.py` in Python
2. Enter the target IP address
3. View open ports in the terminal

## Example

Enter target IP: 8.8.8.8
Scanning 8.8.8.8...
